# GAP REPORT — RRP v0.1.1

## Closed in v0.1.1

- Real Ed25519 signatures
- Device identity bound to public key hash
- Deterministic `aeo_id`
- Single-outcome strict verifier semantics
- Tamper, gap, and fork demo coverage

## Still Open

- TPM quote-backed attestation
- Anti-rollback / monotonic counter
- External anchoring
- Privacy-preserving selective disclosure
- Input trust hierarchy beyond declared provenance

## Honest Assessment

This release is strong enough for public review as a minimal cryptographic provenance protocol. It is not yet a hardware-rooted attestation system.
